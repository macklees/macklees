Microsoft

A collection of consulting work with the following teams: Microsoft 365, Microsoft Launchpad, Microsoft Azure, Microsoft Dynamics 365
Overview
Working as a vendor for Microsoft, I have been a part of teams that delivered web experiences for Microsoft 365 and Microsoft Azure marketing, and conducted user research for products within the Microsoft product line-up (Microsoft Launchpad, Microsoft Dynamics 365).

Projects occurred between July 2018 and July 2019.
My Contributions
For website deliverables, I developed wireframes to express organization of content, flow, and micro-interactions, then produced high-fidelity comps to inform the final design moving into development.

I facilitated global web-based usability studies for Microsoft Launchpad, and conducted user interviews for customers of Microsoft Dynamics 365.
The IT Pro Assessment, a self-serve assessment tool that is helping Microsoft target future investments
IT Pro Assessment
UX/UI Designer
The IT Pro Assessment web experience helped engage over 1,000 attendees at Microsoft Ignite 2018 with a custom digital solution. This self-serve IT Maturity Assessment tool (mobile-centric website) sought to guide IT pros to evaluate their organization's digital maturity. The tool sparked conversations that made Microsoft front-of-mind, and added value by providing an interactive results page to help justify organizational investments for IT in the future.

The project helped Microsoft to capture valuable responses across IT pro roles, org sizes, and geographic locations that can begin to target future investments.
IT Pro Assessment results: a radar chart enabled IT pros to visualize how their organization compared to others at Ignite 2018
Azure Readiness Tool
UX Designer
The Azure Readiness Tool sought to encourage Azure adoption in accordance with biz & tech leaders' business goals and the current state of their business compared to those goals. The tool produced a set of curated resources for ITDMS and BDMS to accelerate the business from current state to vision state, with a push towards Azure adoption.
Microsoft Launchpad
Usability Testing Facilitator
In order to inform the design of additional features to the Microsoft Launchpad tool, a rigorous set of usability studies were conducted across a global user base (Microsoft product sales individuals) to test a core set of tasks and assess user mindset of the tool. Facilitation, coding, and analysis of these usability tests bubbled up to inform design changes that aligned with business priorities.
Microsoft Dynamics 365
UX Designer, User Researcher
Engaged in user research with 20 IT Decision Makers and Business Decision Makers to learn more about how organizations organize and rally to evaluate a new CRM or ERP for their business.
